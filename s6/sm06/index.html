Snapdrop

Ready to Share your local files with your friends?

Snapdrop is a local file sharing solution which completely works in your browser. A bit like Apple's Airdrop, but not only for Apple devices.

Amazing Features

Snapdorp is packed with features and possibilities, making it easy to share everything you
have on you phone with your friends and family

  • Open Source

    App is developed using some of the best security protocols. On top of that, it’s open-source, so developers are continuously improving it.

  • Cross Platform

    Snapdrop works in your browser. A bit like Apple's Airdrop, but not only for Apple devices. Windows, Linux, Android, IPhone, Mac - no problem at all!

  • Easy to Use

    Everything is pretty straightforward. As long as you have both devices connected to the same Wi-Fi network, you can access SnapDrop and send your files.
